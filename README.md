# 🥳 HomeToGather

HomeToGatherApp은 1인가구의 홈파티를 위한 앱 입니다.<br/>
색다른 홈파티를 즐길 수 있도록 초대장과 피드백을 통해 주최자와 참여자의 소통을 도와주는 서비스를 제공합니다.


## :pushpin: Features
- 초대장 생성
- 주최한 파티 관리
  - 공유, 수정, 삭제
  - 참여자 피드백 확인
- 초대된 파티 관리
  - 참여 여부 보내기  
  - 피드백 보내기
  - 주소 복사
-  입장 티켓 애니메이션
